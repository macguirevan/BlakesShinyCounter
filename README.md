# BlakesShinyCounter

Simple Python project that runs the `Shiny_Counter_v2.py` script.

## Overview

This repository contains a small counter script and a JSON configuration file. It's intended to be run locally with Python.

## Files

- `Shiny_Counter_v2.py` — main script
- `shiny_counter_config.json` — JSON configuration
- `requirements.txt` — Python dependencies

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage

Run the script with Python:

```bash
python Shiny_Counter_v2.py
```
